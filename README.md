# 个人面试知识总结

按「知识领域」分目录沉淀的面试笔记仓库。每篇笔记尽量遵循统一结构:**解决问题 → 原理 → 通用模板/套路 → 实战 → 自检清单**。

> 知识不是碎片,而是「**看到题能归类、归类后能套套路、套不上能讲清为什么**」。
